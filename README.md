## Github Page

关于博客搭建，目前最主流的是 Hexo + Next 主题，看了一些教程文章，需要装 Node.js、hexo 等一系列配置，才能实现在本地编写内容后提交到 GitHub Page，但是本身如果对主题要求不高、仅仅是为了记录内容的话，就用官方提供的默认方式写就好了，而且 Setting 里面可以选择不少主题，够用了。
